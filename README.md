# Azure Databases

This repository is a **comprehensive documentation hub** for everything related to **Azure Databases** including migrations, deployments, configurations, and best practices.

The goal is to collect **scripts, documentation, and learnings** across all Azure database services, such as:

- Azure SQL Database
- Azure SQL Database Hyperscale
- Azure SQL Managed Instance  
- Azure SQL VM  
- Azure Arc-enabled SQL Server  
- Azure Database for PostgreSQL Flexible Server
- Azure Database for MySQL  
- Azure Migrate for database discovery and assessment
